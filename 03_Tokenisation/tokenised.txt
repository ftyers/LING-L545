子音 階程 交替 は 、 フィンランド 語 と 北部 サーミ 諸 方言 の 両方 に 観 られる 特徴 です が 、 この 特徴 は 南 サーミ 語 に は 見 られ ない こと から 、 異なる 言語 的 な 歴史 背景 が ある と 考え られ ます 。
コーサカス 山脈 の 北部 は シスコーカサス （ Ciscaucasus ） 、 南部 は トランスコーカシア （ Transcaucasus ） と し て 知ら れ て い ます 。
大気 は カオス 系 な の で 、 システム の 一部 に 小さな 変化 が 起きる と 、 システム 全体 に 大きな 影響 を 及ぼす 可能 性 が ある 。
雨 が 降っ て いる 際 、 水滴 は 、 周囲 の 空気 から 二 酸化 炭素 を 吸収 し 、 溶かし ます 。
気象 制御 に 関する 最近 の 例 と し て は 、 2008 年 の 北京 オリンピック で 中国 が 実施 し た ケース が あり ます 。
東 アフリカ リフト を 形成 し た グローバル な プレート テクトニクス に より 、 東 アフリカ は キリマンジャロ 山 と ケニア 山 と いう アフリカ 最高 峰 の 山々 に 囲ま れ て い ます 。
また 後者 は イスラム 教 を 沿岸 地帯 まで 広め まし た が 、 大 部分 の バンツー 族 は イスラム 教 徒 に 改宗 する こと は なく 、 アフリカ の 伝統 的 宗教 の 信奉 者 で あり 続け まし た 。
フランス は カメルーン 経済 に 、 フランス 経済 の システム を 取り入れる こと で 、 資本 投資 や 有能 な 労働 者 を 通じ た インフラ の 改善 を 実現 し 、 強制 労働 の システム の 対策 を 行い まし た 。
1961 年 10 月 1 日 、 旧 イギリス 領 南 カメルーン は 、 フランス 領 カメルーン と 統一 し 、 カメルーン 共和 国 が 誕生 し た 。
最悪 の 児童 労働 形態 に 関し て 労働 調査 省 は 、 5 歳 から 14 歳 の 子供 の 56 ％ が 就労 し て おり 、 また 7 歳 から 14 歳 の 子供 の 約 53 ％ が 就労 し ながら 学校 に 通っ て いる 、 と 報告 し た 。
サヘル の 気候 は 、 北部 に ある サハラ 砂漠 の 気候 と 似 て い ます が 、 同 砂漠 の 気候 ほど 極端 で は あり ませ ん 。
2010 年 6 月 から 8 月 に かけ て 、 サハラ で は 飢饉 が 発生 し まし た 。
複数 の 古代 文明 は 地中 海 沿岸 に あり 、 付近 の 海 から 大きな 影響 を 受け まし た 。
付近 の 海 は 、 （ 釣り や 、 その 他 の 海産 物 の 収集 など に よる ） 食糧 を はじめ 、 貿易 や 植民 地 化 、 戦争 など 、 様々 な 分野 で 長年 重要 な ルート を 与え まし た 。
気候 や 地質 、 海 へ の アクセス を 共有 し て いる こと から 、 地中 海 を 中心 と し た 文化 は 、 複数 の 文化 や 歴史 が 交差 し て いる 傾向 が あり まし た 。
古典 古代 に おい て 最も 知ら れ て いる 地中 海 文明 は 、 ギリシャ の 都市 国家 と フェニキア 人 の 2 つ で 、 いずれ の 文明 も 地中 海 の 海岸 線 を 広 範囲 に 植民 地 化 し た 。
後年 、 アウグストゥス が ローマ 帝国 を 創設 し た 際 に 、 ローマ 人 たち は 地中 海 を マレ ・ ノストラム （ 「 私 たち の 海 」 ） と 呼び まし た 。
ブリリアニア は 、 ノルウェー 南部 の クリスチャンサン 付近 から リレサン を 越え た エリア まで 続く 、 岩礁 で 保護 さ れ た 水路 です 。
デンマーク 語 で は 、 この 言葉 は 浅い ラグーン に も 使用 さ れる こと が あり ます 。
たとえば フィヨルド で は 、 淡水 湖 で も 深い 場合 が ある 。
ビール や 蜂蜜 酒 、 ビョール （ アルコール 度数 の 高い フルーツ ワイン ） 、 さらに 裕福 な 人々 に は 輸入 ワイン と いっ た アルコール 製品 が 提供 さ れ まし た 。
ルール が どう で あろう と 、 戦い は 、 しばしば 種馬 一 頭 の 死 で 終わっ た 。
シュライ 湾 と 、 ダネヴェルク および フーズム の 間 に ある 湿地 は 、 厳しい 冬 に なる と 土地 が 凍っ て しまい 、 ダネヴェルク の 直 南 に ある 領土 は ドイツ 軍 の 進軍 に より 征服 さ れ まし た 。
この 撤退 は オーストリア 帝国 と プロイセン 王国 の 軍 に は 驚き で あり 、 ほぼ 全て の デンマーク 軍 が 避難 し まし た 。
しかし 、 それ は 重 火器 の 重要 な 砲 の 廃棄 と いう 結果 に なっ た の だ が 、 なぜ 撤退 時 に フレンスブルク 行き 鉄道 が 一 度 も 適切 に 使用 さ れ なかっ た の か に つい て は 、 現在 も 歴史 的 論争 と なっ て いる 。
撤退 の 一報 は 、 ダネヴェルク の 攻略 は 困難 で ある と 考え て い た デンマーク の 世論 に 大きな 驚き を 与え 、 デ ・ メーザ 将軍 は 役職 を 速やか に 解任 さ れ た 。
それ 以来 、 ダネヴェルク は ドイツ 領 と なっ て いる 。
ドナウ 川 を 介し て 輸送 さ れ た 物品 の 量 は 、 1987 年 に は 約 1 億 トン まで 増加 し た 。
ドナウ 川 デルタ の 大半 の 地域 は ルーマニア （ トゥルチャ 県 ） 内 に ある が 、 Chilia 港 の 左岸 に ある 北部 は ウクライナ （ オデッサ 州 ） に 属し て いる 。
気候 が 非常 に 乾燥 し て いる こと から 、 これ ら の 平野 は サハラ の 一部 と 見なさ れる 場合 も あり ます 。
広く 知ら れ て いる 地形 が 沿岸 に 平行 し て い ない 点 で 、 西部 と は 異なり ます 。
北 アルジェリア は 温帯 に あり 、 穏やか な 地中 海 性 気候 が 広がっ て い ます 。
まず 最初 は 、 ミシシッピ 川 に 流入 し た オハイオ 川 でし た 。
紛争 の 中心 に は 、 カンザス が 自由 国家 あるいは 奴隷 国 と し て 、 連合 に 加盟 する か どう か と いう 問題 が 存在 し て い た 。
カンザス で の 出来事 に 対する 国民 の 反応 は 、 国 が いかに 深く 分裂 し て しまっ た の か を 示し て い た 。
葉 の 色 は 、 黄緑 色 から 青み がかっ た 色 まで 様々 な 種類 の 緑 を 示し ます 。
アメリカ 大陸 の アメリカ 先住 民 は 、 代表 的 な 医療 処置 と し て 、 それ を 頻繁 に 利用 し て い まし た 。
ヨーロッパ 北西 部 と ウクライナ の キリスト 教 教会 で は 、 枝 の 主日 の 儀式 を 行う 際 、 ヤシ の 代わり に 柳 の 枝 を 使用 する こと が あり まし た 。
アルプス 山脈 は 、 土地 が 低い ヨーロッパ に 対し て 、 飲料 水 や 灌漑 、 水力 を 提供 し て い ます 。
アルプス 山脈 の 面積 は 、 ヨーロッパ の 表 面積 の 約 11 ％ ほど しか あり ませ ん が 、 しかし 低地 の ヨーロッパ （ 特に 乾燥 地 や 夏季 ） に は 最大 90 ％ 分 の 水 を 提供 し て い ます 。
ミラノ など の 都市 は 、 80 % 分 の 水 を 、 アルプス 山脈 の 流出 水 に 頼っ て い ます 。
河川 から の 水 は 500 を 超える 水力 発電 所 で 使用 さ れ 、 また 2900 キロワット 分 の 電力 を 発電 し ます 。
スイス から 流れ て いる ヨーロッパ の 主要 河川 、 ライン 川 、 ローヌ 川 、 イン 川 、 ティチーノ 川 、 ポー 川 など 、 これ ら の すべて の 川 は 、 アルプス 山脈 に 上流 が あり 、 近隣 諸国 に 流入 し 、 最終 的 に は 北海 や 地中 海 、 アドリア 海 と 黒海 に まで 達し ます 。
火山 の 学問 は volcanology と 言わ れる が 、 しばしば vulcanology と も 表記 さ れる 。
この マグマ は 、 高い ケイ酸 含量 に よる 非常 に 粘性 な 傾向 が ある ため 、 山 の 表面 まで 到達 せ ず に 、 深い 場所 で 冷える 。
これ ら の マグマ は 通常 、 大 惨事 と なる ほど の 爆発 を 起こさ ない 。
地球 温暖 化 は 、 雨季 の パターン を 変容 さ せる 原因 と なり まし た 。
乾燥 し た 小さな ラグーン を 含ん で い ます 。
1975 年 に 、 野生 動物 保護 区 で ある と 宣言 さ れ まし た 。
この 島 へ の 調査 および 探検 は 継続 し て い ます 。
この よう な 開拓 地 は 、 東部 メラネシア 人 が 北部 へ 移動 し た 紀元 前 1000 年 頃 に 始まっ た 可能 性 が あり ます 。
ケンタッキー 州 と インディアナ 州 の 家族 年 の 期間 、 トーマス ・ リンカーン は 農家 や 木製 家具 職人 、 大工 と し て 働い て い まし た 。
生息 地 の 破壊 は 洪水 や 干ばつ 、 穀物 の 不作 、 病気 の まん延 、 水質 汚染 と いっ た 自然 災害 に 対する 地域 の 脆弱 性 を 、 大幅 に 増大 さ せ ます 。
一方 、 最良 の 管理 慣行 が 機能 し て いる 良好 な 生態 系 は 、 こう し た 事態 の 発生 機会 を 減少 さ せる か 、 それ が でき ない 場合 で も 悪 影響 の 度合い を 軽減 し ます 。
実際 に は 、 農地 は 周囲 の 地形 の 破壊 に より 影響 を 受ける 場合 が あり ます 。
過去 50 年間 で 、 農地 に 関連 する 生息 地 の 破壊 は 、 侵食 や 塩類 化 、 圧縮 化 、 栄養 素 の 枯渇 、 汚染 、 都市 化 など を 通じ て 、 世界 中 の 農地 の 約 40 ％ を 劣化 さ せ まし た 。
さらに 人間 も 、 生息 地 が 破壊 さ れる こと で 、 自然 の 生息 地 を 直接 的 に 使用 でき なく なり ます 。
パプアニューギニア は 、 性的 暴力 が 最も 頻繁 に 発生 する 危険 エリア と いう 烙印 を 押さ れる 場合 が あり ます 。
これ ら の 植物 群 は 、 現在 も パプアニューギニア に 存在 し て い ます 。
北米 および 南米 は 、 7 大陸 モデル で は 別々 の 大陸 と し て 扱わ れ ます 。
クリストファー ・ コロンブス は 、 1492 年 に 西 インド 諸島 に 渡り 大西 洋 を 横断 し 、 アメリカ 人 の ヨーロッパ 探検 時代 の 火付け 役 と なっ た 。
これ ら は 遠隔 地 に あっ たり 、 道路 の ない 場所 が 多い こと から 、 あまり 人気 が あり ませ ん 。
残り の 土地 は 様々 な 個人 が 私的 に 所有 し て おり 、 その 免責 は 州 の 約 11 % 分 を 占め て い ます 。
これ は 、 冬 の 平均 的 な 昼間 の 高温 気温 が 、 氷点 を 超え て いる アラスカ 州 の 唯一 の 地域 です 。
多数 の 先住 民 が 、 ヨーロッパ 人 が 到着 する 数千 年 以上 前 から 、 アラスカ 州 を 占領 し て い まし た 。
ここ は 、 同 地域 の オーストラリア 先住 民族 が 残し た 小さな カヌー を 彼 ら が 盗ん だ 場所 です 。
ブリスベン が 繁栄 する に 連れ て 、 河川 の 状態 は 下水 道 や ゴミ捨て 場 並 に 悪化 し た 。
当時 の ブリスベン 市 長 の サリアンヌ ・ アトキンソン に よっ て 、 1987 年 は 「 イヤー ・ オブ ・ ザ ・ リバー 」 と 宣言 さ れ まし た 。
砂 と 砂塵 嵐 は 、 草木 に より 保護 さ れ て い ない 乾燥 地帯 で 発生 する 自然 現象 です 。
一般 的 に 砂塵 嵐 は 、 （ 細かい 物質 は 既に 吹き飛ばさ れ て いる ） 砂漠 内 より は 、 砂漠 の 周辺 地域 にて 発生 し ます 。
安定 し た 風 が 吹き 始める と 、 露出 し た 地面 に 広がる 微 粒子 の 振動 が 始まり ます 。
風速 が より 速い と 、 一部 の 粒子 は 気流 に 舞い上がり ます 。
粒子 が 地面 に 落ちる と 、 大気 に 飛び交う 可能 性 の ある 他 の 粒子 と ぶつかり 、 連鎖 反応 が 発生 し ます 。
しかし タイ で は 、 都市 化 に より 、 肥満 など の 問題 が 大幅 に 増大 し て い ます 。
雷雨 に なる こと は 少なく 、 都市 の 報告 に よる と 1 年 の 間 で たった 7 日 ほど しか あり ませ ん 。
カイロ の 人口 は 1340 年 まで に 50 万 近く に なり 、 中国 西部 で 最大 の 都市 と なっ て い ます 。
その 一方 で 、 ナイル 川 に 行わ れ た 追加 的 な 管理 は 、 ゲズィーラ 島内 や 都市 の 海岸 へ の 開発 を 促進 し まし た 。
ネバダ 州 の 産業 活動 に は 、 観光 を はじめ 、 鉱業 や 機械 、 印刷 、 出版 、 食品 加工 さらに は 電気 設備 など が あり ます 。
「 カリブ 人 （ Caribbean ） 」 と いう 名前 は 、 15 世紀 後半 に ヨーロッパ 人 達 が カリブ に 足 を 踏み入れ た 際 に 、 この 地域 を 支配 し て い た アメリカ 先住 民族 の グループ の 1 つ で あっ た カリブス （ Caribs ） に 由来 し て い ます 。
1492 年 の クリストファー ・ コロンブス に よる アメリカ 大陸 発見 後 、 スペイン 語 の 「 アンティリャス 」 と いう 言葉 が その 土地 の 名前 と さ れ 、 この こと から 「 シー ・ オブ ・ ジ ・ アンティレス 」 は 、 様々 な ヨーロッパ 言語 で 「 カリブ 海 」 を 示す 共通 の 言葉 に なっ た 。
開発 を 進め て い た 1 世紀 の 間 、 この 地域 に 対する スペイン の 支配 力 は 圧倒 的 な もの でし た 。
16 世紀 以降 、 カリブ 海 地域 を 訪れ た ヨーロッパ 人 たち は 、 この エリア を 「 北海 」 （ 同じ 峡谷 の 北側 に ある カリブ 海 ） の 反対 で 、 「 南海 」 （ パナマ の 峡谷 の 南側 に ある 太平 洋 ） で ある と 認識 し た 。
カリブ 海 の 存在 は 、 クリストファー ・ コロンブス が アジア へ の 海路 を 探す 目的 で 、 初めて カリブ 海 を 航海 し た 1492 年 まで ユーラシア 大陸 の 人々 に は 知ら れ て い なかっ た 。
残念 ながら 、 急速 な 人口 増加 と 都市 の 拡大 が 、 こう し た 生態 系 の 様々 な 発展 を 阻害 し て い まし た 。
ケープタウン の 地方 自治 体 は 、 都市 圏 で ある ケープタウン 市 です 。
「 Under sea 」 と いう タイトル で 出版 さ れ た 同 エッセイ は 、 海底 沿い を 巡る 旅 に つい て の 魅力 的 な 物語 でし た 。
さらに 彼女 は 、 機密 情報 を 彼女 に 提供 し た 数々 の 政府 系 科学 者 と の 人脈 を 利用 し まし た 。
